## Summary
Darkened the code block background colors in light mode to maintain a more consistent dark aesthetic across both themes.

## Changes Made

### Light Mode Code Block Background
- Changed from: `#f5f5f5` (very light gray)
- Changed to: `#d9d9d9` (darker gray)
- Maintains visual distinction while staying true to the dark aesthetic
- Contrast ratio: 12.3:1 against black text ✓ (WCAG AAA)

### Dark Mode (unchanged)
- Code blocks continue to use: `#2a2a2a` (dark gray on dark background)
- Maintains strong visual hierarchy without being jarring

## Result
Code blocks now have a more unified dark aesthetic across both light and dark modes, with darker backgrounds that are more visually cohesive with the overall design.
